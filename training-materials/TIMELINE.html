Apa itu Playwright CRX?

Chrome Extension untuk merekam actions browser dan mengexport ke kode Playwright (TypeScript/JavaScript).
Bisa record assertions juga!

Langkah-langkah Install:

  1. Buka link Chrome Web Store di atas
  2. Klik "Add to Chrome" → "Add extension"
  3. Icon Playwright CRX muncul di toolbar

Fitur Utama:

  1. Record - Record actions otomatis

Session 6: Script Optimization 60 menit
1
Robust selectors (getByRole, getByTestId) Wajib

Selectors:

// ❌ Bad
page.click('.active > div:nth-child(2)');

// ✅ Good
page.getByRole('button', { name: 'Submit' });
page.getByTestId('submit-btn');
page.getByLabel('Email').fill('test@test.com');
2
Page Object Model (POM) Wajib

Page Object:

// pages/LoginPage.ts
export class LoginPage {
  constructor(private page: Page) {}

  async login(user: string, pass: string) {
    await this.page.fill('#username', user);
    await this.page.fill('#password', pass);
    await this.page.click('button');
  }
}

// Use in test
test('login', async ({ page }) => {
  const loginPage = new LoginPage(page);
  await loginPage.login('admin', 'pass');
});
